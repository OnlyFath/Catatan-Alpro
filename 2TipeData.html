2.1. Tipe Data (Data Types)

Tipe data adalah klasifikasi jenis data yang menentukan nilai apa yang dapat disimpan dan operasi apa yang dapat dilakukan pada data tersebut. Python adalah bahasa pemrograman yang dinamis, sehingga Anda tidak perlu mendeklarasikan tipe data secara eksplisit, Python akan mengenalinya secara otomatis berdasarkan nilai yang diberikan.

Python memiliki beberapa tipe data dasar yang umum digunakan:

  • int: bilangan bulat, misal 10, -3
  • float: bilangan desimal, misal 3.14, -0.001
  • str: string atau teks, misal "Halo", 'Python'
  • bool: boolean, hanya True atau False
  • list: daftar elemen, misal [1, 2, 3]
  • tuple: daftar elemen yang tidak bisa diubah, misal (1, 2, 3)
  • dict: pasangan key-value, misal {"nama": "Andi", "umur": 20}
