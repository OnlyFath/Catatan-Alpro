Bahasa Pemrograman Python

Apa itu Python?

Python adalah bahasa pemrograman tingkat tinggi yang dikembangkan oleh Guido van Rossum dan pertama kali dirilis pada tahun 1991. Python dikenal karena sintaksnya yang sederhana dan mudah dipahami, sehingga sangat cocok untuk pemula.

Keunggulan Python

  • Mudah dipelajari: Sintaks yang jelas dan ringkas.
  • Multiguna: Digunakan untuk web development, data science, machine learning, otomasi, dan lain-lain.
  • Banyak pustaka (library): Memiliki ekosistem pustaka yang luas untuk berbagai kebutuhan.
  • Komunitas besar: Dukungan komunitas yang aktif dan banyak sumber belajar.

Struktur Dasar Program Python

Contoh program Python sederhana untuk menghitung luas persegi panjang:

In[]:
# Program menghitung luas persegi panjang

# Input panjang dan lebar dari user
panjang = float(input("Masukkan panjang: "))
lebar = float(input("Masukkan lebar: "))

# Hitung luas
luas = panjang * lebar

# Tampilkan hasil
print("Luas persegi panjang adalah:", luas)

Penjelasan

  • Fungsi input() digunakan untuk menerima input dari pengguna.
  • Fungsi float() mengkonversi input string menjadi angka desimal.
  • Operator * digunakan untuk perkalian.
  • Fungsi print() menampilkan output ke layar.

Kesimpulan

Pemrograman adalah proses penting dalam dunia teknologi yang dimulai dari pemahaman algoritma sebagai langkah-langkah penyelesaian masalah. Python sebagai bahasa pemrograman yang mudah dipelajari sangat cocok untuk pemula yang ingin memulai belajar pemrograman. Dengan memahami algoritma dan menguasai Python, seseorang dapat mengembangkan berbagai aplikasi dan solusi teknologi.
