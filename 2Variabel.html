2.2. Variabel (Variables)

Variabel adalah nama yang digunakan untuk menyimpan data dalam memori komputer. Variabel memungkinkan kita untuk menyimpan, mengubah, dan menggunakan kembali data tersebut dalam program.

  • Nama variabel harus dimulai dengan huruf atau underscore (_)
  • Tidak boleh menggunakan spasi atau karakter khusus
  • Case sensitive (nama dan Nama berbeda)

Contoh kode variabel:

In[4]:
# Mendefinisikan variabel
nama = "Budi"
umur = 25
tinggi = 172

print(nama)
print(umur)
print(tinggi)

Budi
25
172

Quiz Variabel

Diberikan tiga variabel berikut:

IntData = 10
FloatData = 20.5
StrData = "Halo Namaku Fathir"

Soal:

  1. Tampilkan data dari ketiga variabel tersebut.
  2. Tampilkan tipe data masing-masing variabel tersebut.
  3. Tampilkan data pada variabel StrData.
In[8]:
IntData = 20
FloatData = 4.5
StrData = "Halo Namaku Fathir"

# 1. Tampilkan data ketiga variabel
print("IntData:", IntData)
print("FloatData:", FloatData)
print("StrData:", StrData)

IntData: 20
FloatData: 4.5
StrData: Halo Namaku Fathir
In[6]:
# 2. Tampilkan tipe data masing-masing variabel
print("Tipe IntData:", type(IntData))
print("Tipe FloatData:", type(FloatData))
print("Tipe StrData:", type(StrData))

Tipe IntData: <class 'int'>
Tipe FloatData: <class 'float'>
Tipe StrData: <class 'str'>
In[7]:
# 3. Tampilkan data pada variabel StrData
print("Isi StrData:", StrData)

Isi StrData: Halo Namaku Fathir
